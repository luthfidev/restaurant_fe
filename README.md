# Frontend Restaurant React JS

***

## Dependencies

* Node Js
* react-router-dom
* axios
* jasonwebtoken(JWT)
* universal-cookie
* use-global-hook
* semantic-ui-css
* semantic-ui-react
* react-semantic-toasts
  
***

## Installing

Clone the repo, install dependencies, and start the Frontend server locally.

```shell
git clone https://github.com/luthfidev/restaurant_fe.git
cd muhiLibrary_backend
create .env
vim .env
yarn install
yarn start 
```
***
